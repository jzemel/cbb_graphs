# Comedy Bang Bang Graphs
Data visualization for CBB podcast, guests and characters. See who made appearances over time, when characters overlap, which episodes had lots of guests, which episodes had guests playing a lot of characters.

Links out to the cbb fandom wiki and to earwolf to play the eps. All data from (and credit to) the fandom API and the wiki editors.

Served from github pages, accessible at https://cbbgraphs.com OR https://jzemel.github.io/cbb_graphs
